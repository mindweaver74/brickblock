# 🧱 벽돌깨기 REBORN

> 아이템으로 다이나믹하게 벽돌을 부수세요! 공이 늘어나고, 불이 붙고, 폭탄이 터진다!

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/brickbreak-reborn](https://mindweaver74.github.io/brickbreak-reborn)

---

## 🎮 게임 방법

1. 화면을 **터치/드래그**해서 패들을 좌우로 이동
2. 공이 벽돌에 맞으면 HP가 감소, 0이 되면 벽돌 파괴
3. 벽돌 파괴 시 **아이템이 떨어짐** → 패들로 받으면 효과 발동!
4. 모든 벽돌을 깨면 다음 스테이지로 진행
5. 공을 놓치면 목숨 1개 감소, 목숨이 0이 되면 게임 오버

## ✨ 특징

- 💥 **다이나믹 물리** — 패들 각도에 따라 공 방향 조절 가능
- 🧱 **벽돌 HP 시스템** — HP에 따라 색상 변화 (밝음→회색→어두움)
- 📈 **스테이지 난이도** — 스테이지마다 벽돌 HP 증가
- 🌟 **파티클 이펙트** — 폭발, 아이템, 공 잔상 화려한 효과
- 📳 **화면 흔들림** — 폭발 시 박진감 있는 진동 효과
- 🏆 **최고 기록 저장** — 브라우저에 자동 저장
